    I don't mind text-based technical references, but they aren't for everyone. So a graphic SQL cheat sheet may help Confirm the Power of Visuals in eLearning. In the spirit of collaboration the downloadable versions are available in PDF and PNG formats on my GitHub repository.
    The flowchart covers many common T-SQL data manipulation commands that my team uses regularly in a format that can help quickly build statements from left-to-right with fields, rows, values, etc. that are color coded for easy reference.
